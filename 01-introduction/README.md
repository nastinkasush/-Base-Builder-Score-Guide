# 01 — Введение

## Что такое Builder Score?

Builder Score (также называют Builder Rank) — это репутационная оценка от **Talent Protocol**, которая показывает, насколько ты активный и полезный участник экосистемы **Base**.

### Почему это важно?
- Еженедельные награды в ETH (Builder Rewards)
- Приоритет при раздаче Basenames
- Видимость для грантов и партнёрств
- Высокий статус в сообществе

**Score обновляется каждые 14 дней.**
