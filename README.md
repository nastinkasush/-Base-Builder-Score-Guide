## Быстрый старт

1. Создай [Talent Passport](https://app.talentprotocol.com)
2. Получи **Human Checkmark**
3. Купи Basename (.base.eth)
4. Добавь проект в [Onchain Registry](https://buildonbase.deform.cc)
