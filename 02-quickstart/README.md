# 02 — Быстрый старт (за 1-2 дня)

### Чек-лист для быстрого старта (40+ очков)

- [ ] Зарегистрироваться на [talent.app](https://talent.app/)
- [ ] Получить **Human Checkmark**
- [ ] Купить Basename (.base.eth)
- [ ] Подключить GitHub
- [ ] Задеплоить 1 простой контракт
- [ ] Сделать 20–30 транзакций на Base
- [ ] Добавить проект в Registry

**Ожидаемый результат:** 45–60 очков после первого обновления.
